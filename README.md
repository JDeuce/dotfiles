## Intro ##

This is my dotfiles

## Basic Install ##

 1. Clone the repo.

        * git clone github.com/JDeuce/dotfiles.git ~/dotfiles

 2. Install symlinks to all the dotfiles in our home directory

        * cd ~/dotfiles
        * gmake install # [1]

        [1]: if your system has GNU make installed as make you can
        just run make instead of gmake
